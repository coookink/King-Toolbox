# 🤖 OpenCode (编程特工) 指南

## 1. 这是什么？(What is this?)
OpenCode 是开源界的 "Devin" —— 一个**会写代码的 AI 员工**。
它与其他 AI 工具（如 Cursor）最大的区别是：**它是自主的**。
你给它一个任务（比如“帮我写个贪吃蛇游戏”），它会自己：
1.  创建文件。
2.  写代码。
3.  运行报错。
4.  读错误日志。
5.  自己修复 Bug。
6.  直到跑通为止。

## 2. 安装教程 (Mac/Linux)

### 第一步：一键安装
OpenCode 非常极客，没有图形界面，完全在**终端**里运行。
打开终端，粘贴以下命令：

```bash
# 官方安装脚本
curl -fsSL https://opencode.ai/install | bash
```

### 第二步：配置环境
安装后，你需要告诉它用哪个大脑（模型）。
在终端输入：
```bash
opencode config
```
按提示选择模型供应商（推荐 Anthropic Claude 3.5 Sonnet 或 OpenAI GPT-4o），并粘贴 Key。

### 第三步：开始干活
让它帮你写代码：

```bash
# 启动并进入交互模式
opencode
```
或者直接给任务：
```bash
opencode "请在当前目录下创建一个 snake_game.py，实现贪吃蛇游戏"
```

## 3. 核心快捷键 (TUI 操作)
OpenCode 的界面像黑客帝国一样酷（TUI），你需要掌握几个键：
*   **`Ctrl + C`**: 它是停不下来的，如果你觉得它跑偏了，按这个强行打断。
*   **`Tab`**: 自动补全命令。

## 4. 常见问题
*   **Q: 它一直循环报错修不好怎么办？**
    *   A: 这种现象叫 "Loop of Death"。按 `Ctrl+C` 停止它，然后给它更具体的指引（比如：“不要用 A 库，改用 B 库”）。
*   **Q: 它可以读我的本地文件吗？**
    *   A: 可以。它默认拥有你当前运行目录的**读写权限**。所以**请务必在一个安全的、独立的文件夹里运行它**，以免它不管是删了你的重要文件。
