# 🎨 UI/UX Pro Max Skill 终极指南

## 1. 这是什么？
这是你的**AI 首席设计师**。
如果你是独立开发者，经常发愁："这按钮该用什么颜色？"、"这布局怎么不好看？"。
这个库为 AI (Trae/Cursor) 植入了**几百条专业设计规则**。一旦启用，AI 写的代码不仅能跑，而且**好看**。

## 2. 核心能力
*   **自带审美**: 内置 67+ UI 风格 (玻璃拟态、极简、赛博朋克...)。
*   **智能配色**: 输入 "医疗App"，自动推荐专业且让人信任的蓝色系，而不是随便找个蓝。
*   **代码生成**: 直接生成 SwiftUI 或 React 代码。

## 3. 小白使用教程

### 第一步：准备工作 (安装)
确保你电脑安装了 Node.js。
在终端运行：
```bash
# 全局安装 CLI 工具
npm install -g uipro-cli
```
*(注：如果报错找不到命令/网络错误，说明该工具源可能在维护，请直接使用下方的“手动复制法”。)*

### 第二步：注入灵魂 (Init)
在你的项目文件夹下运行：
```bash
uipro init --ai trae
```
这会自动生成一个 `.cursorrules` 文件。Trae 会自动读取它，从此 Trae 就“懂”设计了。

### 第三步：手动复制法 (备选方案 - 推荐)
如果不想装工具，最稳妥的方法是：
1.  打开本仓库 `modules/ui-ux-pro-max/knowledge.md` (或类似核心规则文件)。
2.  复制全部内容。
3.  在 Trae 的对话框里说：
    > "请记住以下设计规范，接下来的所有 UI 代码都要严格遵守："
    > (粘贴内容)

## 4. 实战指令 (直接复制给自己用)

**场景：设计一个登录页**
复制这段话给 Trae：

> **Role**: 资深 UI 设计师
> **Task**: 帮我写一个 SwiftUI 的登录页面。
> **Requirements**:
> 1.  **风格**: 使用 Glassmorphism (毛玻璃) 风格。
> 2.  **配色**: 科技感深色模式 (Dark Tech)。
> 3.  **组件**: 包含用户名、密码、漂亮的渐变色登录按钮。
> 4.  **动效**: 按钮点击要有缩放反馈。

## 5. 常见问题
*   **Q: 生成的界面还是很难看？**
    *   A: AI 需要明确的风格关键词。试着加上具体词汇，如 "Modern", "Clean", "Apple-style", "Padding: 20px"。
